# Escopo Academico do MVP

## Finalidade

Este MVP foi criado para apoiar uma apresentacao academica sobre Comercio Exterior e Logistica, simulando o acompanhamento de importacoes maritimas ficticias com destino ao Porto de Santos.

## O Que o Protótipo Faz

- Exibe uma base mockada de processos ficticios.
- Permite filtrar por status academico, pais de origem e risco academico.
- Calcula indicadores simples de quantidade de processos, conteineres, valores estimados e armazenagem estimada.
- Mostra graficos e tabela para fins didaticos.

## Limites Declarados

- Os dados sao 100% ficticios.
- O app nao se conecta ao Siscomex.
- O app nao se conecta a Receita Federal.
- O app nao se conecta ao MDIC.
- O app nao se conecta ao Portal Unico.
- O app nao se conecta a terminais portuarios, armadores, agentes de carga, despachantes ou bancos.
- O app nao cria API real.
- O app nao usa credenciais, tokens, certificados digitais ou chaves secretas.
- O app nao informa NCM.
- O app nao informa aliquotas.
- O app nao determina necessidade de LI, LPCO ou qualquer autorizacao oficial.
- O app nao interpreta regra aduaneira, fiscal, cambial, tributaria, sanitaria, ambiental ou regulatoria.

## Premissas Academicas

- Prazos, valores, empresas, navios, portos e observacoes sao inventados.
- A classificacao de risco e apenas uma etiqueta didatica.
- O status academico nao equivale a status oficial de processo aduaneiro.
- Os valores FOB, frete, seguro, CIF e armazenagem sao estimativas ficticias para demonstracao.

## Fora de Escopo

- Integracao com sistemas governamentais.
- Consulta oficial de declaracoes, licencas, documentos ou pagamentos.
- Calculo tributario real.
- Classificacao fiscal de mercadorias.
- Emissao de documentos.
- Autenticacao de usuarios.
- Banco de dados persistente.
- API externa ou interna.

## Uso Correto

Use este projeto apenas para demonstrar uma ideia de painel academico, fluxo de dados mockados e visualizacao simples em Streamlit.
